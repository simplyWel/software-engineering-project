# Projeto TDD e Gitflow

Este repositório segue o padrão Gitflow simplificado com Desenvolvimento Dirigido por Testes (TDD).

## Estrutura do Projeto
* `src/`: Arquivos fonte.
* `include/`: Arquivos de cabeçalho (`.hpp`).
* `test/`: Arquivos de teste.
* `bin/`: Binários gerados (ignorados pelo Git).
